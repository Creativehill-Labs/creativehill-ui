# 🎨 Creativehill UI (CUI)

Creativehill UI는 Creativehill Design System(CDS)에 기반하여 만들어진 React Component Library입니다.

더 나은 UX(User Experience)와 DX(Developer Experience)를 위한 메이커들의 노력으로 탄생🐣했습니다.

Creativehill의 프로덕트는 [**component-driven**](https://componentdriven.org) 방식으로 설계하는 것을 권장하며, 원자(atomic) 단위로 시작하여 페이지(page) 단위로 끝나는 개발 프로세스를 지향합니다.

## 지향점
1. 모든 인풋은 `react-hook-form` 라이브러리와의 쉬운 결합이 가능하도록 설계
2. 타입스크립트 기반으로 작성
3. Mui의 설계 방식에 영감을 받음

## 사용법

좌측의 사이드바를 통해 컴포넌트 목록에 접근할 수 있으며, `스토리`를 통해 각 컴포넌트의 코드와 작동예제를 확인할 수 있습니다.
